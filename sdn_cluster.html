SDN Graph

The matrix below examines the user population of the Student Doctor Network allopathic medical school application threads for the 2010-11 application cycle. Each row and column represents a school's thread, and each data point is a measure of overlap between the user populations of those two threads. The darker the color, the more common users those two schools have on SDN's forums. The matrix is clustered by Spearman correlation. Note that schools tend to cluster roughly by ranking/competitiveness, and geography.
